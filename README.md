### 结构说明

- assets 资源
- bean 结构返回得 data 结构
- components 辅页面
- manager 数据处理（UserManager 为本地数据处理）
- param 接口预处理（接口、参数）
- router 路由
- store 【待定】
- utils 工具类
- views 主页面
- App.vue（壳）
- main.ts (程序入口)
- permission.ts (路由拦截)
- request.ts (网络请求顶层处理)
